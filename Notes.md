## Formulas
* circumference: ``2*pi*r``
* area: ``pi*r**2``
* 2 circles intersect when ``(x1 - x2)**2 + (y1 - y2)**2 <= (R1 + R2)**2``
* a point is inside a circle if ``(x1 - x2)**2 + (y1 - y2)**2 < R**2``
